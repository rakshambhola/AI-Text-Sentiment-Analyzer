# AI Text Sentiment Analyzer

An NLP-based system that classifies input text into **Positive**, **Negative**, or **Neutral** sentiment using Python and NLTK's VADER sentiment analyzer.

## Features

* Classifies sentiment of user-provided text
* Uses NLP techniques with a pre-trained VADER lexicon
* Fast and lightweight execution
* Beginner-friendly 

## Technologies Used

* Python
* NLTK
* VADER Sentiment Analyzer

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/rakshambhola/AI-Text-Sentiment-Analyzer.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the program:

```bash
python sentiment_analyzer.py
```

## Example

**Input:**

```
I really enjoyed working on this project.
```

**Output:**

```
Sentiment: Positive
```
